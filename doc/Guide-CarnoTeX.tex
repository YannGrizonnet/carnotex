\section{Présentation générale}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

CarnoTeX est un ensemble de packages personnels destinés à produire des documents mathématiques et pédagogiques cohérents.

\Cbox{CarnoDef}{Architecture générale}{
\begin{itemize}
  \item \pkg{carnotex.sty} : mise en forme générale ;
  \item \pkg{carnomath.sty} : commandes mathématiques et environnements ;
  \item \pkg{carnopeda.sty} : outils pédagogiques et gestion des versions ;
  \item \pkg{carnolegacy.sty} : compatibilité avec les anciens documents.
\end{itemize}
}

\Cbox{CarnoProp}{Options de chargement}{
\begin{itemize}
  \item \cmd{usepackage}\argu{carnotex} : noyau seul ;
  \item \cmd{usepackage}\opt{math}\argu{carnotex} : noyau + mathématiques ;
  \item \cmd{usepackage}\opt{peda}\argu{carnotex} : noyau + pédagogie ;
  \item \cmd{usepackage}\opt{legacy}\argu{carnotex} : noyau + compatibilité ;
  \item \cmd{usepackage}\opt{all}\argu{carnotex} : tout charger.
\end{itemize}
}

\Cbox{CarnoRem}{Niveaux de document}{
Les niveaux permettent de produire plusieurs versions d'un même document.

\begin{itemize}
  \item \cmd{level}\argu{eleve} : version distribuée aux élèves ;
  \item \cmd{level}\argu{prof} : version professeur ;
  \item \cmd{level}\argu{notes} : version professeur enrichie avec notes, TODO, sources et temps.
\end{itemize}
}

\begin{commande}{df}{carnomath}

\Syntaxe{
\syntaxe{\df{contenu}}
\syntaxe{\df*{contenu}}
}

\Arguments{
\begin{itemize}
  \item \texttt{contenu} : texte de la définition ;
  \item version étoilée : définition encadrée.
\end{itemize}
}

\Usage{
Présente une définition. La version étoilée utilise un cadre coloré tout en conservant la numérotation.
}

\Exemple{
\df{
Une fonction affine est une fonction de la forme $f(x)=ax+b$.
}

\df*{
Une fonction linéaire est une fonction affine vérifiant $b=0$.
}
}

\end{commande}

La somme de deux fonctions continues est continue.
